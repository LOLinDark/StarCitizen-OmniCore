## Support the Project

OmniCore is built and maintained by **LOLinDark** as a passion project for the Star Citizen community.

If you find it useful, here are some ways you can help:

### ⭐ Star the Repo

The simplest way to show support — [star OmniCore on GitHub](https://github.com/LOLinDark/StarCitizen-OmniCore).

### 🐛 Report Bugs

Found something broken? [Open an issue](https://github.com/LOLinDark/StarCitizen-OmniCore/issues) on GitHub.

### 🤝 Contribute

Pull requests are welcome. See the [Developer Guide]({{ '/developer-guide/' | relative_url }}) for architecture and contribution info.
